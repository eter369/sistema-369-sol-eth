# Sistema 369 — Terminal educativa SOL/ETH

Guía interactiva de un solo archivo sobre un sistema de rotación **SOL/ETH** medido en ETH, de *bitCODE Trading Labs · Omraam Group*.

Todo vive en `index.html` — sin build, sin dependencias más allá de Google Fonts y Chart.js por CDN.

## Qué incluye

- **Panel** — ratio SOL/ETH editable, distancia al próximo tramo, estado del ciclo y el gráfico maestro con las bandas del sistema (pico, tramos −10% / −15%, invalidación 0.0355, MA25 y marcadores de compra/venta).
- **En simple** — la tesis y el ciclo de 4 pasos como diagrama animado.
- **Las reglas** — las 5 reglas + tabla de señales ejecutable + las 6 mejoras pro.
- **Simulador** — calculadora de tramos, simulador de ciclo con checklist de disciplina, donut de asignación y barra de exposición (nunca supera el 30%).
- **Bitácora** — journal de ciclos con métricas automáticas, escalera de ETH y export/import JSON (localStorage).
- **Riesgos** — los 4 riesgos, medidor de exposición y la frase de cierre.
- **Ticker en vivo (Binance)** — precios de BTC, ETH y SOL por WebSocket, con el ratio SOL/ETH derivado en tiempo real.

## Uso

Abre `index.html` en el navegador, o visita el sitio publicado con GitHub Pages.

> **Nota:** el ticker usa la API pública de Binance; si tu red o país la bloquea, mostrará "sin conexión" y el resto de la app sigue funcionando.
